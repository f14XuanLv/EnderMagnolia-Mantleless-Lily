# EnderMagnolia Mantleless Lily

Lily appearance modifications for Ender Magnolia.

## Installation

### 1. Download the desired mod files

`Mantleless_Lily.zip` or `Impurity-Revealing_Lily.zip`

### 2. Navigate to the game's Paks folder

```
\steam\steamapps\common\ENDER MAGNOLIA\EnderMagnolia\Content\Paks\
```

### 3. Place the mod files in the Paks folder

```
📁 Paks\
  ├─ EnderMagnolia-Windows.pak
  ├─ EnderMagnolia-Windows.ucas
  ├─ EnderMagnolia-Windows.utoc
  ├─ global.ucas
  ├─ global.utoc
  ├─ ─────────────────────────────── (original game files above)
  └─ 📁 Mantleless_Lily\ (mod files)
      ├─ Mantleless_Lily_0_P.pak
      ├─ Mantleless_Lily_0_P.ucas
      └─ Mantleless_Lily_0_P.utoc
```

### 4. Launch the game to see the changes.
